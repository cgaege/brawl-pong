# Brawl Pong

A Brawl Stars-themed Pong game playable entirely in the browser — no install, no dependencies, just open `index.html`.

## Features

- **Three game modes** — Solo (vs. bot), Duo (2 players), Co-Op (2 players vs. bot)
- **Brawler avatar selection** — pick your character from real Brawl Stars portraits before each match
- **Destructible obstacles** — crates in the arena that take multiple hits to break
- **Shield power-up** — catch the floating shield to block the ball from reaching your goal
- **VS intro screen** — cinematic countdown before each game starts
- **Serve mechanic** — ball sticks to your paddle; launch it when ready
- **Avatar reactions** — brawlers bounce or shake on score events
- **Visual effects** — ball trail, particles, shockwave rings, screen shake, score flash

## Controls

| Action | Solo / Co-Op (P1) | Duo (P2) | Co-Op (P2) |
|---|---|---|---|
| Move up | `W` or `↑` | `↑` | `↑` |
| Move down | `S` or `↓` | `↓` | `↓` |
| Pause | `P` / `Escape` | | |

## How to play

1. Open `index.html` in any modern browser.
2. Select a game mode and (if applicable) a difficulty.
3. Pick your brawler avatar.
4. Survive the VS intro and launch the ball — first to 7 points wins.

## Development

Single-file vanilla HTML/CSS/JS — everything lives in `index.html`.
